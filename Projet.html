Île de Groix

Pour ce projet d'atelier professionnel, je devais créer et présenter un site web sur une île de Bretagne.
Pour notre groupe c'est tombé sur l'île de Groix, située au sud ouest de Lorient
Les objectifs de ce projet était de présenter l'histoire de l'île de Groix, ses balades à faire là-bas, les horraires des marées du jour et les risques sur cette île

Mathématiques BTS SIO

Dans le cadre de l'épreuve pratique de BTS SIO, il fallait réaliser une application web ou lourde que nous devons présenter devant un jury et modifier devant eux.
Comme j'adore les math et aider les autres j'ai décider de créer une application web de gestion de cours de math de BTS SIO qui regroupe tout les cours de math de BTS SIO avec des simulateurs pour aider.
Ces cours sont sous forme de carte et les utilisateurs peuvent créer leur propre carte de cours. De plus pour accéder à ce cours les utilisateurs doivent s'inscrirent et se connecter.
Ces utilisateurs comme les cartes qui créer sont regroupe dans une base de donnée sur wampserver64
